# 顔認証 × GPS連携 勤怠管理アプリ

リモートワークでも正確な勤怠管理ができるように、  
**顔認証・GPS・打刻履歴管理・勤務時間の自動集計・グラフ可視化** を組み合わせた  
スマホ対応 Web アプリを個人開発しています。

AWS Amplify Hosting を利用し、  
**認証あり版（本番用）** と **認証なし版（ポートフォリオ用）** をブランチごとに運用しています。

## 🛠 使用技術

### フロントエンド
- React + Vite  
- Tailwind CSS  
- Recharts（グラフ可視化）  
- Amplify UI / Material-UI（UI強化予定）

### バックエンド
- AWS Amplify（Auth / GraphQL API / Hosting）
- DynamoDB（勤怠データ管理）
- S3（顔認証画像保存）

### 顔認証
- Python（face_recognition）
- FastAPI（認証 API）

### その他
- Geolocation API（GPS取得）

## ✨ 特徴

### 🔐 顔認証 × GPS × 勤怠管理
- 顔認証による本人確認で不正打刻を防止  
- GPS で位置情報を取得し、外出先でも正確な打刻が可能  
- 出勤・退勤の打刻区分を明確に記録  
- 打刻履歴を日本時間で一覧表示（日時・位置・住所付き）

### 🗂 管理者ページ
- 社員名 / 打刻種別 / 日時 / 場所 / 顔写真 をテーブル表示  
- 社員別・日付別の検索・フィルタ機能  
- 本番環境では GraphQL + S3 + GPS の実データを使用  
- ポートフォリオ版ではリアルなダミーデータで動作

### 📊 ダッシュボード（NEW）
- **月毎の勤務時間合計を自動計算**  
- **残業時間（8時間超過分）を自動算出**  
- **ユーザー別の勤務時間を棒グラフで可視化**  
- **残業時間を折れ線グラフで表示**  
- UUID のままでは見づらいユーザーIDを、**名前に変換して表示**  
- 実務レベルの勤怠分析 UI を再現
