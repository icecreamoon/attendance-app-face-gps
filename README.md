# 顔認証 × GPS連携 勤怠管理アプリ

📌 概要  
リモートワークでも正確な勤怠管理ができるように、**顔認証・GPS・打刻履歴管理**を組み合わせたスマホ対応 Web アプリを個人開発しています。  
AWS Amplify Hosting を利用し、**認証あり版（本番用）と認証なし版（ポートフォリオ公開用）**をブランチごとに運用しています。  
コストを抑えつつ、実用性・セキュリティ・UI/UX のバランスを重視した設計を目指しています。

---

## 🛠 使用技術
- フロントエンド: React + Vite, Tailwind CSS  
- バックエンド: AWS Amplify (Auth, GraphQL API, Hosting)  
- データベース: DynamoDB（勤怠データ管理）  
- 画像保存: S3（顔認証画像）  
- 顔認証: Python（face_recognition） + FastAPI（API連携）  
- その他: Geolocation API（GPS取得）  
- UI: Amplify UI, Material-UI（導入予定）  

---

## ✨ 特徴
- 顔認証による本人確認で不正打刻を防止  
- GPSで位置情報を取得し、外出先でも正確な打刻が可能  
- 出勤・退勤の打刻区分を明確に記録  
- 打刻履歴を日本時間で一覧表示（日時・位置・住所付き）  
- **管理者ページを実装**  
  - 社員名 / 打刻種別 / 日時 / 場所 / 顔写真 をテーブル表示  
  - 検索・フィルタ機能（社員別・日付別）を搭載  
  - ポートフォリオ用ではダミーデータで動作、本番用では GraphQL + S3 + GPS で実データを管理  
- 認証あり版と認証なし版をブランチで切り替え可能  
- スマホ対応のレスポンシブ UI  

---

## ✅ 実装済み機能
- ログイン認証（Amplify Auth / Cognito）  
- 顔認証 → 打刻処理（FastAPI + Python）  
- GPS位置情報の取得と履歴への紐づけ  
- 出勤・退勤の打刻区分の追加  
- 打刻履歴の一覧表示（日時・位置・住所付き）  
- 撮影画像の保存（S3）  
- 管理者ページ（勤怠データ一覧 + 検索機能）  
- 認証なし版（ポートフォリオ用）の公開
