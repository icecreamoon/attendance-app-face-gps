# 顔認証 × GPS連携 勤怠管理アプリ

## 📌 概要

リモートワークでも正確な勤怠管理ができるように、**顔認証・GPS・打刻履歴管理**を組み合わせたスマホ対応のWebアプリを個人開発しています。  
AWSの無料枠を活用し、コストを抑えながら実用性とセキュリティを両立した設計を目指しています。

## 🛠 使用技術

- **フロントエンド**：React + Vite, Tailwind CSS  
- **バックエンド**：AWS Cognito（認証）, DynamoDB（勤怠データ）, S3（画像保存）  
- **顔認証**：Python（face_recognition） + FastAPI（API連携）  
- **その他**：Geolocation API（GPS取得）, AWS Amplify（GraphQL連携）

## ✨ 特徴

- 顔認証による本人確認で不正打刻を防止  
- GPSで位置情報を取得し、外出先でも正確な打刻が可能  
- 出勤・退勤の打刻区分を明確に記録  
- 打刻履歴を日本時間で一覧表示（日時・位置・写真付き）  
- 撮影画像は管理者が確認可能（セキュリティ強化）  
- スマホ対応のレスポンシブUIで、どこでも使える

## ✅ 実装済み機能

- ログイン認証（Cognito）  
- 顔認証 → 打刻処理（FastAPI + Python）  
- GPS位置情報の取得と履歴への紐づけ  
- 出勤・退勤の打刻区分の追加  
- 打刻履歴の一覧表示（日時・位置・写真付き）  
- 撮影画像の保存（S3 or ローカル）  
- 顔認証結果に応じた打刻制御（不一致時は打刻不可）
